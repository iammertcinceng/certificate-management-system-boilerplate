# Dashboard Template

> **Disclaimer:** Bu proje tamamen eğitim ve yetkinlik sergileme amaçlı jenerik bir versiyondur; herhangi bir ticari kuruluşun aktif sistemini veya verisini temsil etmez.

This is a [Next.js](https://nextjs.org) enterprise dashboard structured project. It showcases modern web architecture, robust database integration, and clean business logic patterns.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
