# 🛡️ CERTIFICATE VERIFICATION AND MANAGEMENT SYSTEM BOILERPLATE

> **⚠️ Disclaimer:** Bu proje, kurumsal seviyedeki sertifikasyon ve doğrulama iş akışlarını sergilemek amacıyla geliştirilmiş, **anonimleştirilmiş bir referans mimaridir.** Herhangi bir ticari kuruluşun özel verisini, ticari sırlarını veya aktif sistemlerini temsil etmez. Tüm iş mantığı ve UI bileşenleri genel kullanım standartlarına göre yeniden yapılandırılmıştır.

This is a [Next.js](https://nextjs.org) enterprise dashboard structured project. It showcases modern web architecture, robust database integration, and clean business logic patterns for secure digital credentialing.

---

## 🚀 Key Engineering Highlights

Bu proje, karmaşık onay mekanizmalarını yönetmek için inşa edilmiş bir **"Software Architect"** portfolyo örneğidir:

* **Multi-Party Verification:** Akreditör, kurum ve son kullanıcı arasındaki güven zincirini yöneten hiyerarşik yapı.
* **RBAC (Role-Based Access Control):** Next.js Server Actions ve Middleware ile sıkılaştırılmış yetkilendirme katmanı.
* **Relational Excellence:** PostgreSQL & Drizzle ORM kullanılarak kurgulanmış, yüksek tutarlılığa sahip veri modeli.
* **Type-Safe Development:** Zod ile şema doğrulama ve uçtan uca TypeScript desteği.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js 14+ (App Router), Tailwind CSS, Lucid Icons
* **Backend:** Server Actions, Zod Validation
* **Database:** PostgreSQL (Relational)
* **ORM:** Drizzle ORM
* **Security:** JWT-based Auth / Middleware Protection
